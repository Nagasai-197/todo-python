# To-Do Manager

## Overview
A console-based To-Do Manager built using Python that allows users to create, update, delete, and manage tasks with priority levels.

## Features
- Add Tasks
- Update Tasks
- Delete Tasks
- Mark Tasks Complete
- Filter Tasks by Priority
- JSON-based Storage

## Technologies Used
- Python
- JSON

## How to Run
python main.py
